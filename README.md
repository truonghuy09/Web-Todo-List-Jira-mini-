# Web Todo List – Jira Mini

## Giới thiệu
Dự án Web Todo List mô phỏng hệ thống Jira Mini, phục vụ môn Đồ án chuyên ngành.

## Mục tiêu
- Quản lý công việc theo Kanban Board
- Áp dụng Sprint, Role-based Access Control
- Có thể mở rộng thành đồ án tốt nghiệp

## Công nghệ dự kiến
- Frontend: ReactJS
- Backend: ASP.NET Core
- Database: SQL Server
